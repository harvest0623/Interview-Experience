## 四、手撕代码
### 1. 判断两个矩形是否相交

#### 代码：
```JavaScript
function fn(rect1, rect2) {
    return !(
        rect1.x2 <= rect2.x1 ||
        rect1.x1 >= rect2.x2 ||
        rect1.y2 <= rect2.y1 ||
        rect1.y1 >= rect2.y2
    )
}

const rect1 = {x1: 100, y1: 100, x2:300, y2:300};
const rect2 = {x1: 200, y1: 200, x2: 400, y2: 400};
console.log(fn(rect1, rect2));
```

#### 思路：（只有这四种情况，矩形才不相交）
- 矩形1 的右边界在 矩形2 的左边
- 矩形1 的上边界在 矩形2 的下边
- 矩形1 的左边界在 矩形2 的右边
- 矩形1 的下边界在 矩形2 的上边
